PRIMARY EXPORT

rkph_data.js

The single data file. Contains all students, marks, fees, attendance, notices, timetable. Replace this one file on your website — both portal and admin panel update automatically.

1

Make any changes in this panel.

2

Download rkph_data.js.

3

Replace old rkph_data.js on website. Done.

admin_panel.html

Saves the admin panel itself. Use this to keep a backup of the panel with all your settings. For data updates, use rkph_data.js above.

1

Backup before major changes.

2

Download admin_panel.html.

3

Store safely as backup.

Keep all 4 files in the same folder on your website:
rkph_data.js   student_portal.js   student_portal.html   student_portal.css   index.html   admin_panel.html
